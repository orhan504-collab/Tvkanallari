# TurkBox TV - Android Box Canlı TV
Bu proje, Türkiye'deki ücretsiz ulusal kanalları Android TV ve Box cihazlarında izlemek için geliştirilmiştir.

## Özellikler
- Kumanda (D-Pad) uyumlu arayüz.
- GitHub üzerinden güncellenebilir kanal listesi.
- ExoPlayer ile kesintisiz HLS yayını.
